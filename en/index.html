Makiden Lite

An Innovative Calculator with History and Simplicity

Makiden Lite is a revolutionary calculator app that combines ease of use with powerful functionality. It maintains the familiar operations of a traditional calculator while introducing a unique paper roll display for calculation history.

Recognized as one of the Best iPhone Apps of 2010 by AppBank, it has gained widespread user appreciation. Since version 7.0, it has incorporated a decimal arithmetic engine, enhancing calculation accuracy. Make your daily calculations easier and more precise.

Use cases:
Shopping expense calculations, household budget verifications, business expense calculations, educational math tool, and more.

Key Features

1. Paper Roll Display

Displays the calculation process like a paper roll, allowing you to check it at any time. You can easily review past calculation history by scrolling.

2. Standard/Advanced/Note Keyboard Switching

We've implemented three keyboard modes: Standard, Advanced, and Note. Users can easily switch between them based on their preferences and calculation needs.

  • Standard: Provides simple and easy-to-use basic calculator functions
  • Advanced: Extended keyboard with more advanced features
  • Note: Provides note-taking functionality to save and manage calculation results

3. Flexible Decimal Point Modes

Three decimal point modes are available:

  • F (Free): No limit on decimal places
  • 0 (Zero): Integer-only display
  • 2 (Euro mode): Display up to two decimal places

Easily switchable with the "dp/VAT" button (in Advanced mode).

4. VAT Calculation Feature

By long-pressing the "dp/VAT" button in Advanced mode, you can set the VAT rate. After setting, simply switch to +VAT mode with the "dp/VAT" button to automatically calculate with added tax.

5. Scan Function

Use the "SCAN" button in Advanced mode to scan numbers from receipts or price lists and automatically input them. This is particularly useful for correctly inputting large numbers, such as for tax returns.

6. Advanced Calculation Functions

  • Square Root Calculation: Easily calculate square roots with the "√" button.
  • Sign Inversion: Switch between positive and negative numbers with the "±↻" button.
  • Backspace Function: Delete from the last digit in case of input errors (Advanced mode).
  • Restore Function: Restore numbers deleted by backspace (Advanced mode).
  • C/AC Function: Pressing the "C/AC" button resets the input or calculation result. Long-pressing clears all content on the paper roll and saves it to Note.

7. Copy Function

Use the "copy" button in Advanced mode to copy the calculation history displayed on the paper roll to the clipboard.

8. Total Calculation Function

The "=/Σ" button displays normal calculation results. In Advanced mode, long-pressing calculates the sum of all calculation results displayed on the paper roll.

9. Note Function

Save calculation results as notes for later reference or editing. Saved notes are grouped by date for easy management.

10. User-Friendly Interface

  • Intuitive operation method
  • Clear display
  • Dark mode support
  • Easy switching between Standard/Advanced/Note modes
