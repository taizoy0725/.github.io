Rolled Paper Calculator Lite

An Innovative Calculator with History and Easy-to-Use Features

Rolled Paper Calculator Lite is an innovative calculator app that combines ease of use with functionality. While maintaining the operability of traditional calculators, it features a function that displays calculation history like a roll of paper.

Selected as the Best iPhone App of 2010 by AppBank, it has been supported by many users. From Ver7.0, a decimal calculation engine has been incorporated, improving calculation accuracy. You can perform daily calculations more easily and accurately.

Use cases: Calculating expenses while shopping, checking household accounts, calculating business expenses, as a calculation tool for learning, and more

Main Features

2. Flexible Decimal Point Modes

Equipped with three decimal point modes:

  • F (Free): No limit on the number of decimal places
  • 0 (Zero): Integer display only
  • 2 (Euro mode): Display up to two decimal places

Easily switchable with the "dp/VAT" button.

3. Consumption Tax (VAT) Calculation Function

Long-press the "dp/VAT" button to set the consumption tax rate. After setting, just use the "dp/VAT" button to enter +VAT mode for automatic tax-inclusive calculations.

4. Scan Function

Use the "SCAN" button to scan numbers from receipts or price lists and automatically input them. Particularly effective for correctly entering large numbers, such as for tax returns.

5. Advanced Calculation Functions

  • Square Root Calculation: Easily calculate square roots with the "√" button.
  • Sign Inversion: Switch between positive and negative values with the "±↻" button.
  • Backspace Function: Delete the last digit in case of input errors.
  • Restore Function: Restore deleted numbers with the backspace function.
  • C/AC Function: Press the "C/AC" button to reset the current input or calculation result. Long-press to clear all content on the paper roll.

6. Copy Function

Copy the calculation history displayed on the paper roll to the clipboard with the "copy" button.

7. Total Calculation Function

The "=/Σ" button displays the normal calculation result. Long-press to calculate the sum of all calculation results displayed on the paper roll.

8. User-Friendly Interface

  • Intuitive operation
  • Clear display
  • Dark mode support
